# 📝 Minimalist Todo List App

A sleek, lightweight task management application built from scratch using **Vanilla JavaScript, HTML5, and CSS3**. This project focuses on a clean user experience and robust state management without relying on heavy frontend frameworks.

## ✨ Features

* **Full CRUD Functionality:** Seamlessly create, read, complete, and delete tasks.
* **Data Persistence:** Utilizes browser `localStorage` to save your tasks locally—your data survives page refreshes and browser restarts.
* **Smart Filtering:** Switch between **All**, **Active**, and **Completed** views to easily manage your workload.
* **Progress Tracker:** Dynamic "items remaining" counter that updates instantly as you check off tasks.
* **Fully Responsive:** Beautifully optimized for mobile, tablet, and desktop viewports using CSS Flexbox and Grid.

---

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Custom properties/variables, Flexbox, Grid)
* **Logic:** Vanilla JavaScript (ES6+ features like array methods, arrow functions, and modules)
* **Storage:** Web Storage API (`localStorage`)
